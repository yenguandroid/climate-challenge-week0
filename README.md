# Climate Challenge Week 0

## Setup Instructions

1. Clone repository:
   git clone <repo-url>

2. Create virtual environment:
   python -m venv venv

3. Activate environment:
   venv\Scripts\activate  (Windows)

4. Install dependencies:
   pip install -r requirements.txt
